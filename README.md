# 🛡️ Cybersecurity Write-Ups & Hands-on Labs

Bem-vindo ao meu repositório de portfólio técnico em Segurança Ofensiva e Defensiva. Aqui documentei de forma minuciosa a resolução de laboratórios práticos, simulando cenários reais de testes de invasão (Pentest).

O objetivo deste repositório é demonstrar minhas habilidades práticas em mapeamento de superfície de ataque, análise de vulnerabilidades, exploração, pós-exploração e proposição de medidas mitigadoras de segurança.
